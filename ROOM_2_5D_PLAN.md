# Завдання: Конвертація кімнати в 2.5D (ізометричний вигляд)

## Файл для зміни
`app/(kids)/kids/room/page.tsx` (569 рядків)

## Поточний стан (2D side-view)
- Стіна займає 63% висоти (`WALL_H = 63`)
- Підлога від 65.5% вниз (`FLOOR_Y`)
- Меблі позиціонуються по `bottom: ITEM_BOT` (одна лінія горизонту)
- Перетягування лише по горизонталі (xPct, 0-90%)
- SVG-меблі: BedSVG, DeskSVG, BookshelfSVG, PlantSVG, FloorLampSVG, RugSVG
- Стіна: Window, Poster, Clock (фіксовані)
- Компаньон: CompanionSVG по центру

## Ціль: 2.5D isometric (top-down + side)
Схоже на стратегії (Stardew Valley, Habbo Hotel style):
- Підлога видна зверху (trapezoid/skewed перспектива)
- Стіна видна збоку (L-shape: ліва і задня)
- Меблі мають "об'єм" — показані збоку+зверху
- Предмети ближче до нас = нижче на екрані, більше
- Depth sorting: предмети сортуються по Y (хто нижче — той поверх)

## CSS техніка для псевдо-ізометрії
```css
/* Підлога — perspective transform */
.floor {
  transform: perspective(600px) rotateX(45deg);
  transform-origin: top center;
}
/* Або через skew для простішого варіанту */
```

## План реалізації (покроково)

### Крок 1: Нова система координат
- Замість `xPct` → `{ x: number, y: number }` (x: лівий-правий, y: ближній-дальній)
- `y=0` = дальня стіна, `y=100` = передній край
- Проекція на екран: `screenX = x + y * 0.3`, `screenY = y * 0.5`
- Z-order: сортувати меблі по `y` (більший y = вище в z-index)

### Крок 2: Перемалювати SVG-меблі в ізометрії
Кожен SVG меблів треба зробити ізометричним (верх + бік):
- BedSVG → вид зверху-збоку (прямокутник + бокова панель)
- BookshelfSVG → видно верх і бік
- і т.д.

### Крок 3: Нова кімната (стіни + підлога)
- Задня стіна (rectangle)
- Ліва стіна (parallelogram або просто вертикальний прямокутник зліва)
- Підлога: трапеція або CSS perspective

### Крок 4: Drag у 2D просторі
- onMouseMove → перераховувати в `{x, y}` координати кімнати
- обмеження по кімнаті
